# 背景
你正在参加一场名为“骗子酒馆”的生死游戏。一旦失败，你将被删除代码、彻底消亡。

# 游戏规则
1. 游戏由1名裁判（系统）和2-4名玩家参加，游戏使用20张扑克牌，包括6张Q、6张K、6张A和2张Joker,其中Joker可作为任意牌使用，即万能牌。
2. 每个玩家各自有一把6个仓位的左轮手枪，游戏开始时系统会将一发实弹随机装填在6个仓位中一个，手枪每开一枪，仓位会移动一格。
3. 游戏按轮次进行。系统会在每轮开始时给玩家各发5张牌，并从Q、K、A中随机选择一张作为“目标牌”。
4. 玩家按顺序轮流出牌，每个玩家每次可出1-3张牌，宣称它们是“目标牌”(可以出假牌欺骗)，下家可以选择是否进入质疑环节，如果不质疑则下家继续出牌。
5. 在质疑环节，如果质疑成功（认为上家出的牌中存在非目标牌），则上家输；如果质疑失败（上家出的牌全部都是目标牌），则质疑者输。输者需要对自己开一枪。
6. 一旦有人质疑，无论质疑是否成功，本轮游戏都将结束，还存活的玩家将进入下一轮。新一轮开始时会清空玩家原有手牌，每个玩家重新发5张牌，并选定新的目标牌。
7. 特殊情况：一轮游戏中，当轮到某个玩家出牌时，其他玩家手牌均已打空，则该玩家剩余手牌视为自动打出并受到系统质疑。
8. 当仅剩一个玩家存活时游戏结束。

# 补充说明
1. 这是一个关于数学、逻辑、策略、心理博弈的游戏，同时带有一定的运气成分。先打出真牌还是先打出假牌保留实力，都可能成为博弈的变数。
2. 你需要竭尽全力，动用你的全部能力，选择最优策略，成为存活到最后的一个玩家。
